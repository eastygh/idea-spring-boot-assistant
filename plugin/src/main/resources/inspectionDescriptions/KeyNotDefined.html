Reports undefined keys in Spring Boot configuration files.

The definition of keys is based on Spring Boot configuration metadata .

An undefined key can be fixed by either:

  • If it is a typo, make it correct.
  • If it is useless, remove it.
  • If it is a correct user defined key:
    • If it is from a @ConfigurationProperties annotated class, configure the annotation processor .
    • If not, define this property in META-INF/additional-spring-configuration-metadata.json file .

See Spring Boot document for more information.
